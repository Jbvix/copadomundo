Como usar o mapa

O que é isto?

Rede de conhecimento da Seleção Brasileira e adversários, gerada a partir do vault Obsidian. Cada bolinha é uma nota; cada linha é um link [[ ]] entre notas.

Nem tudo se conecta

Dois nós só se ligam se houver relação no vault. Brasil não liga a todos os 86 nós — apenas aos vizinhos diretos (times, jogadores, partidas citadas na nota).

Cores por tipo

  • Time
  • Jogador
  • Partida
  • Edição
  • Técnico
  • Clube

Modo foco

Toque em um nó: vizinhos diretos ficam visíveis; o resto esmaece. Setas verdes = conexões diretas. Toque no fundo ou no chip Foco: … para limpar.

Visualização 2D

  • Arraste para mover o mapa · pinça para zoom
  • Busque por nome ou filtre por tipo
  • Botão centraliza o Brasil

Visualização 3D

Botão alterna para o grafo em três dimensões. Arraste para girar · scroll para zoom · clique nos nós como no 2D. Recomendado em telas maiores; no celular pode consumir mais bateria.
